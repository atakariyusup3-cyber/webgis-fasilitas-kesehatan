# WebGIS Fasilitas Kesehatan

WebGIS Fasilitas Kesehatan adalah aplikasi berbasis web yang menampilkan **peta persebaran fasilitas kesehatan** seperti Puskesmas, Klinik, Rumah Sakit, dan Apotek menggunakan **Laravel & Leaflet.js**.  
Aplikasi ini memudahkan masyarakat menemukan fasilitas kesehatan terdekat dengan cepat.
